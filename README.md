# 🌤️ Weather API App

A responsive weather web application built using **HTML, CSS, and JavaScript** that provides real-time weather information, hourly forecasts, and a 5-day forecast using the **Open-Meteo REST API**.

---

## 📖 Project Overview

Weather API App allows users to search for any city and instantly view current weather conditions, temperature, humidity, wind speed, hourly forecast, and a 5-day weather forecast. The application features a modern, responsive interface and fetches live weather data using the Open-Meteo API.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Current temperature
- 🤗 Feels Like temperature
- 💧 Humidity
- 🌬️ Wind speed
- 🕒 Hourly weather forecast
- 📅 5-Day weather forecast
- 🏙️ Dynamic city background
- 📱 Responsive design
- ⚠️ Error handling for invalid city names
- 🌐 Live weather data using REST API

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Install the **Live Server** extension.
4. Open `index.html`.
5. Right-click and select **Open with Live Server**.
